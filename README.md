# 📱 برنامج إدارة مخزون محل الهواتف

نظام شامل لإدارة مخزون محل الهواتف المحمولة مع واجهة عربية سهلة الاستخدام.

## 🌟 المميزات

### 📦 إدارة المخزون

- إضافة وتعديل وحذف المنتجات
- تتبع الكميات والمخزون
- تنبيهات المخزون المنخفض
- إدارة الفئات والعلامات التجارية
- رفع صور المنتجات

### 💰 إدارة المبيعات

- إنشاء فواتير البيع
- حساب الضرائب والخصومات
- طرق دفع متعددة
- طباعة الفواتير (عادية وحرارية)

### 🛒 إدارة المشتريات

- فواتير الموردين
- تحديث المخزون التلقائي
- تتبع تكاليف الشراء

### 👥 إدارة العملاء والموردين

- قاعدة بيانات العملاء
- معلومات الموردين
- تاريخ التعاملات

### 📊 التقارير والإحصائيات

- تقارير المبيعات اليومية والشهرية
- تقارير المخزون
- تصدير Excel
- رسوم بيانية تفاعلية

### 🔐 نظام المستخدمين

- مستويات صلاحيات متعددة
- تسجيل دخول آمن
- إدارة المستخدمين

## 🚀 التشغيل المحلي

### المتطلبات

- Python 3.8+
- pip

### التثبيت

1. **استنساخ المشروع:**

```bash
git clone https://github.com/robotbouba9/store-.git
cd store-
```

2. **تثبيت المتطلبات:**

```bash
pip install -r requirements.txt
```

3. **إنشاء قاعدة البيانات:**

```bash
python init_db.py
```

4. **تشغيل التطبيق:**

```bash
python run_web.py
```

5. **فتح المتصفح:**

```
http://localhost:5000
```

### بيانات الدخول الافتراضية

- **اسم المستخدم:** admin
- **كلمة المرور:** Admin@123

## 🌐 النشر على Render

### الإعداد التلقائي

المشروع مُعد للنشر التلقائي على Render:

1. **ربط المستودع:** اربط مستودع GitHub بـ Render
2. **إعدادات البيئة:** سيتم إنشاء `SECRET_KEY` تلقائياً
3. **قاعدة البيانات:** سيتم إنشاء PostgreSQL تلقائياً
4. **النشر:** سيتم النشر تلقائياً عند كل تحديث

### متغيرات البيئة المطلوبة

```env
APP_ENV=production
SECRET_KEY=your-secret-key
DATABASE_URL=postgresql://...
```

## 📁 هيكل المشروع

```
store/
├── app.py                 # التطبيق الرئيسي
├── config.py             # إعدادات التطبيق
├── database.py           # نماذج قاعدة البيانات
├── forms.py              # نماذج الويب
├── excel_export.py       # تصدير Excel
├── thermal_invoice.py    # الفواتير الحرارية
├── requirements.txt      # المتطلبات
├── Procfile             # إعدادات Heroku/Render
├── render.yaml          # إعدادات Render
├── runtime.txt          # إصدار Python
├── templates/           # قوالب HTML
├── static/             # الملفات الثابتة
├── instance/           # قاعدة البيانات المحلية
└── uploads/            # الملفات المرفوعة
```

## 🛠️ التقنيات المستخدمة

### Backend

- **Flask** - إطار العمل الرئيسي
- **SQLAlchemy** - ORM لقاعدة البيانات
- **Flask-WTF** - نماذج الويب
- **Flask-Login** - إدارة الجلسات

### Frontend

- **Bootstrap 5** - تصميم الواجهة
- **Chart.js** - الرسوم البيانية
- **jQuery** - التفاعل مع الصفحة

### قاعدة البيانات

- **SQLite** - للتطوير المحلي
- **PostgreSQL** - للإنتاج

### التصدير والطباعة

- **ReportLab** - إنشاء PDF
- **OpenPyXL** - تصدير Excel
- **Pillow** - معالجة الصور
